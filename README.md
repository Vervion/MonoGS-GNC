# Getting Started
## Installation
```
git clone https://github.com/Vervion/MonoGS-GNC.git --recursive
cd MonoGS
```

This version was created alongside a Jupyter Notebook which includes all scripts neccessary to run program within Google Colab
## Quick Demo
```
bash scripts/download_tum.sh
python slam.py --config configs/mono/tum/fr1_desk.yaml
```
You will NOT see a GUI
